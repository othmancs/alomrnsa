Multi Warehouse Management on POS Odoo Apps

POS Multi Warehouse Management odoo app helps users to manage point of sale orders with multiple warehouses, users can configure multiple warehouses. Users have also the option to display "Available Quantity or Unreserved Quantity" and selected warehouse stock with selected stock type will be displayed for products in the point of sale screen. While selecting products, users can enter quantities from different warehouses and picking will be created from different warehouses with different quantities. Users have also the option to create picking in "READY" state and allow pos order even product out of stock.

Features

Point of Sale Multi Warehouse

User can configure and display stock from different multiple warehouse.

Display Available or Unreserved Stock Quantity

Display available or unreserved stock quantity in pos screen.

Set Picking In Ready State

User can set point of sale order picking in ready state.

Allow POS Order When Product is Out of Stock

User can create pos order even product out of stock.

POS Multi Warehouse Configuration

Under Point of Sale > Configuration > Point of Sale menu user need to enable "Display stock in pos" option and select warehouse, User also select "Stock Type as Available Quantity or Unreserved Quantity".

User can enable "Set Picking In Ready State" option if wants to create picking in ready state, Also enable "Allow POS Order When Product is Out of Stock" option if wants to sell out of stock product.

Available Product Quantity

User can see available product quantity from multiple warehouse.

POS Product Quantity

User can see product available/unreserved quantity from all selected warehouse as per configuration.

While user select pos product a new pop up will appear, user can see different warehouse and stock from warehouse and enter product quantity and click "Apply" button.

User can see added product with entered quantity from warehouse, User also can see warehouse on pos cart.

POS Receipt with Warehouse

User can see pos order receipt with product and warehouse.

POS Order with Warehouse

User can see created point of sale order with product and warehouse.

POS Order Picking

If user enable "Set Picking In Ready State" option then user can see created picking for pos order in ready state with different warehouse and quantity.

Allow POS Order with Out of Stock Product

User can see pos product with zero available quantity.

User can see out of stock product having zero qty in all warehouse, even user can enter quantity.

User can see on adding out of stock product entered quantity will decreased from stock.
