Product code is a module developed by the Center of Research & Development company. It adds a new *product_code* field to the product model. This code is automatically set by the sequence. Product code sequence generates easily readable codes.

Examples of generated product codes:
  - T-00000001
  - T-00000002
  - \.\.\.
